Tabby Browser
 
CMD + L            Show/Hide URL Bar
CMD + T            New Tab
CMD + [            Previous Tab
CMD + ]            Next Tab
CMD + W            Close Tab
CMD + R            Reload Tab
CMD + Left         Go Back
CMD + Right        Go Forward
CMD + N            New Window
CMD + Shift + W    Close Window
CMD + Q            Quit
